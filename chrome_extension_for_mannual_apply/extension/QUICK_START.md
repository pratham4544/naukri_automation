2. Paste URLs in text box

## Troubleshooting

### Extension Not Working?
- Refresh page (F5)
- Reload extension
- Check console (F12)

### Form Not Filling?
- Click "Fill Current Page" manually
- Check if field labels match memory
- Some sites block auto-fill

### File Upload Failed?
- Some sites require manual upload
- Upload file manually when prompted
- Extension will remember other fields

### Dropdown Not Selecting?
- Try exact option text
- Try shorter match ("2-3" not "2-3 years")
- Select manually if needed
